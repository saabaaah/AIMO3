# AIMO Progress Prize 3 - Competition Overview

## Quick Facts
- **Competition**: AI Mathematical Olympiad - Progress Prize 3
- **Platform**: Kaggle
- **Total Prize Pool**: $2,207,152 + $110,000 (additional prizes)
- **Deadline**: April 15, 2026 (Final Submission)
- **Entry/Team Merger Deadline**: April 8, 2026
- **Current Participants**: 17,077 entrants, 2,532 participants, 2,325 teams

## Objective
Create open-source algorithms and models that can solve olympiad-level math problems written in LaTeX format. Problems range from National Olympiad level to International Mathematical Olympiad (IMO) standard.

## Background
- **AIMO 1 Winner** (July 2024): Project Numina (29/50 problems)
- **AIMO 2 Winner** (April 2025): NVIDIA NemoSkills (34/50 problems)
- This competition features significantly increased difficulty, 5-digit answers (vs 3-digit previously), and enhanced compute resources.

## Problem Specifications
- **Total Problems**: 110 original math problems
- **Public Test Set**: 50 problems
- **Private Test Set**: 50 problems
- **Reference Data**: 10 problems (with solutions provided)
- **Answer Range**: Non-negative integer 0-99999 (5 digits)
- **Domains**: Algebra, Combinatorics, Geometry, Number Theory
- **Format**: Text-only LaTeX (no diagrams)
- **Contamination Risk**: Zero (all problems are original)

## Key Rules & Requirements
1. **Open Source Requirement**: All code and datasets must be made publicly available to qualify for prizes
2. **Code Competition**: Must submit via Kaggle notebooks using the provided Python evaluation API
3. **No Internet**: Notebooks must run with internet access disabled
4. **GPU Options**: H100 GPUs available (AIMO 3 only), or standard Kaggle GPUs
5. **Runtime Limits**:
   - CPU Notebook: <= 9 hours
   - GPU Notebook: <= 5 hours
6. **External Data**: Freely & publicly available external data and pre-trained models allowed
